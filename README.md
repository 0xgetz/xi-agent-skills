# Gumloop Agent — Skills & MCP

This repository is an export of skills and connected MCP server documentation from my Gumloop agent.

## Contents

- **`skills/`** — Core agent skills (skill-creator, gumloop-sdk, trigger-builder, etc.).
- **`trading-skills/`** — 100 trading analysis skills (indicators, patterns, harmonics, risk, backtesting).
- **`mcp-skills/`** — 100 MCP integration skills (GitHub, Slack, databases, vector stores, market-data APIs, more).
- **`telegram-bots/`** — 100 Telegram crypto-alpha & coin-tracker bot skills
  (new-token launch scanners, whale & smart-money trackers, honeypot/rug detectors,
  volume/liquidity/breakout alerts, multi-chain coverage).
- **`MCP_SERVERS.md`** — The MCP integrations actually connected to the live agent.

Each skill is a directory containing a `SKILL.md` with valid YAML frontmatter (`name`, `description`) plus a structured guide body.

- MCP skills are usage guides; live MCP servers must be connected as integrations.
