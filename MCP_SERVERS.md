# Connected MCP Servers

MCP integrations connected to the Gumloop agent at export time.

## apollo
B2B data enrichment. Tools: enrich_organization, enrich_person, find_people, get_organization_job_postings, organization_search.

## brandfetch
Brand data lookup. Tools: enrich_transaction, get_brand, search_brands.

## github
Code & project management. Tools: create/read/update repos, issues, PRs, files, branches, releases, collaborators, stargazers, deployments.

## gpagespeed (Google PageSpeed)
Web performance. Tools: get_core_web_vitals, get_lighthouse_audits, run_pagespeed_analysis.

## reducto
Document parsing & editing. Tools: upload/parse/extract/edit/split/download documents, cancel/list jobs.
